# 프론트 페이지 작업 리스트

---

# GetRequest("")

1. 페이지에 필요한 요청.
ex) 페이지 내부에서 사용되는 비동기 처리 요소

2. 다른 페이지로의 이동.
ex) 현 페이지에서 다른 페이지로 넘어가는 페이지 적인 요소
(주소창으로 이동하는 경우는 배제)

---

# REST URL 리스트: GET (1)

> prefix: /rest   
[GET] /champlist/{lane}?[sort=tier]&[getAmount=0] (챔피언 리스트)   
`{"sort": ["tier", "win", "pick", "ban"], "getAmount": Integer}`   
[GET] /champ/{champName}/rune (챔피언 상세/룬)   
[GET] /champ/{champName}/skill (챔피언 상세/스킬)   
[GET] /champ/{champName}/item (챔피언 상세/아이템)   
[GET] /champ/{champName}/spell (챔피언 상세/스펠)   
[GET] /champ/{champName}/counter (챔피언 상세/카운터)   
[GET] /summoner/{summonerName}/game (소환사 전적/최근 게임)   
[GET] /summoner/{summonerName}/stat (소환사 전적/개인 통계)   

---

# REST URL 리스트: GET (2)

> prefix: /rest   
[GET] /ranking (랭킹)   
[GET] /duolist?[pageNum=1]?[getAmount=0] (듀오 찾기 게시판)   
`{"pageNum": Integer, "getAmount": Integer}`   
[GET] /duo/{postNum} (듀오 찾기 게시글)   
[GET] /login?[id]&[password] (로그인)   

---

# REST URL 리스트 <POST>

> prefix: /rest   
[POST] /join?[userId]&[password]&[lolName="0"] (회원가입)   
[POST] /changepw?[userId]&[password] (비밀번호 변경)   
[POST] /changelol?[userId]&[newlolName] (롤 계정 변경)   
[POST] /newduo?[summonerName] (듀오 게시글 생성)
